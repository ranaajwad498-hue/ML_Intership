# Child Weight Analysis Program

A modular, terminal-based statistical analysis application built using **Python Object-Oriented Programming (OOP)** principles and the **NumPy** scientific computing library. The program instantiates data arrays to efficiently calculate structural weight metrics for pediatric evaluation datasets.

## 🚀 Features & Scope
* **OOP Architecture:** Logic encapsulated neatly inside a reusable `ChildWeightAnalyzer` class definition.
* **Vectorized Computations:** Utilizes highly efficient NumPy array operations instead of standard python loop cycles.
* **Precision Formatting:** All calculated metrics explicitly processed and displayed to 2 decimal places.
* **Bonus Analytics:** Includes calculations for total accumulation vector weights, mid-dataset medians, and logical threshold evaluation filters.
