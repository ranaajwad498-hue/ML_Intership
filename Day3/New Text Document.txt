*Task:*
Build a Child Weight Analysis Program using Python OOP concepts and NumPy.

Create a class named ChildWeightAnalyzer. The class should contain:

*Constructor*
The constructor should receive a list of 10 children’s weights and convert it into a 
NumPy array.
*Example:*
ChildWeightAnalyzer([6.8, 7.5, 8.2, 8.9, 9.4, 9.8, 10.3, 10.9, 11.5, 12.1])

*Methods*
Create separate methods to perform the following operations:

display_weights()
calculate_average()
find_minimum_weight()
find_maximum_weight()
calculate_standard_deviation()
calculate_total_weight()
calculate_median_weight()
count_children_above_average()
display_report()

*Requirements:*
Create an object of the ChildWeightAnalyzer class.
Store the weights of 10 children in a NumPy array.
Use class methods to calculate:
Average weight
Minimum weight
Maximum weight
Standard deviation
Display all results with 2 decimal places.
Keep the code clean and properly organized.
Do not write all logic directly outside the class.

*Bonus Task:*
Add methods to calculate:
Total weight of all children
Median weight
Number of children whose weight is greater than the average weight
