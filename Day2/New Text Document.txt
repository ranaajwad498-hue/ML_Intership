*Task*

Build a Child Information Management System using Python Classes.

*Requirements*

Create a Child class with the following attributes:

Child ID
Child Name
Age
Gender
Parent/Guardian Name
Contact Number
Address

Your application should display the following menu:

===== Child Information Management System =====

1. Add Child
2. View All Children
3. Search Child
4. Update Child Information
5. Delete Child
6. Exit

When a user enters 1, the program should allow the user to add child information and store it in the list.
When a user enters 2, the program should show the stored children's information

Do the same process for options 3, 4, 5, and 6.
Make a repo for this task and push it to GitHub.
