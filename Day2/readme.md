# Child Information Management System

A command-line interface (CLI) application built using core **Python OOP (Object-Oriented Programming)** principles. This system allows users to manage child health and guardian records through standard CRUD (Create, Read, Update, Delete) operations, storing data dynamically using Python classes and lists.

## 📚 Core Concepts Practiced
* **Python Syntax & Structures:** Implementing control flow loops and conditional branching.
* **Object-Oriented Programming (OOP):** Creating dedicated blueprint templates using Python `class` structures, constructors (`__init__`), and instance methods.
* **Data Persistence:** Using runtime memory tracking with Python lists to manage multiple object instances.
* **Virtual Environments:** Isolating project dependencies cleanly using built-in Python tooling (`venv`).
